What this plan refuses to duplicate

Keep v4. Presentation work does not create a v5 cache or make valid recipes regenerate.
Keep one route. The recipe page remains the owner of timers, fallback, edit guards, and cooking completion.
Keep text safe. Typed Svelte segments replace raw HTML and loose substring marking.
Keep old prose exact. Sentence layout never rewrites words; one long cached sentence stays long.
Keep serving changes local. The current quantity projection remains instant and provider-free.
