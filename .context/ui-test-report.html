Failures found and fixed

Failed in browser · fixed afterward

Desktop AI agent did not reliably reopen or restore focus

Closing left focus on BODY; a queued native dialog close event could also close a freshly reopened panel. The fix now waits for the trigger to remount, restores focus explicitly, and ignores stale close events.

Failed in browser · fixed afterward

Two-second timer was labeled “1 minute”

The chip rounded every sub-minute duration up to one minute. It now renders the exact clock value and uses a matching localized accessible label.

Failed in browser · fixed afterward

Shared muted and primary text missed contrast targets

The light theme primary color and low-opacity utility text produced serious contrast findings on several routes. The shared theme now uses a darker primary and an accessible opacity floor.
